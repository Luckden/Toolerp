# Enterprise Meta-Model Research

This workspace investigates whether enterprises are best modeled as hierarchies, graphs, layered systems, federated systems, or hybrids.

Current stance:

- Enterprises are best treated as multi-dimensional semantic graphs with layered views.
- Hierarchies still exist, but they are only one relationship type among many.
- Governance, lifecycle, funding, and technical dependency should never be collapsed into the same tree.
